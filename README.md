# CyberSentinel-AI


AI-Powered Cyber Threat Intelligence and Security Copilot Platform.

## Overview

CyberSentinel-AI is an advanced cybersecurity platform that combines Machine Learning, Deep Learning, Large Language Models, Threat Intelligence, and Anomaly Detection to identify, analyze, and explain cyber threats.

## Features

* Network Intrusion Detection
* Attack Classification
* Anomaly Detection
* Threat Intelligence Retrieval
* AI Security Analyst
* Incident Report Generation
* Security Dashboard
* RAG-Based Cyber Knowledge Base
